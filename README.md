# Утилиты

Сборник небольших полезных утилит в одном окне. Оформление — как в программах
**Senior Hub** (тёмно-зелёная шапка, светло-зелёный фон, плитки со скруглением
и тенью, подвал с версией и настройками, локализация RU/EN).

Программа работает в фоне (значок в области уведомлений), поэтому горячие
клавиши действуют, даже когда окно закрыто.

## Утилиты

### ⌨️ Раскладка — исправление текста, набранного не в той раскладке

Выделите криво набранный текст и нажмите **Pause/Break** — раскладка исправится
прямо на месте (EN ↔ RU). Направление определяется автоматически: если в
выделении есть кириллица — переводим RU→EN, иначе EN→RU.

Примеры:
- `Цштвщцы` → `Windows`
- `ghbdtn` → `привет`
- `Руддщ` → `Hello`

Плитка показывает состояние (**Включено / Выключено**) — нажмите на неё, чтобы
включить или отключить горячую клавишу.

> Технически: одна системная горячая клавиша через `RegisterHotKey` — программа
> **не** перехватывает все нажатия и не является кейлоггером.

### 📷 Снимок → текст (OCR)

Распознаёт текст на экране и копирует его в буфер обмена (как ABBYY Screenshot
Reader). Нажмите плитку — экран затемнится, выделите мышью область с текстом.
Распознанный текст сразу окажется в буфере обмена; в углу появится уведомление.
**Esc** — отмена.

Распознавание выполняет встроенный движок Windows (`Windows.Media.Ocr`) — без
внешних зависимостей. Язык распознавания выбирается в настройках: **Авто** (по
языкам системы, читает и RU, и EN), **Русский** или **English**.

> Требуются языковые компоненты OCR в Windows. Обычно русский и английский уже
> установлены. Проверить: Параметры → Время и язык → Язык → нужный язык →
> Параметры → «Распознавание текста».

## Настройки (⚙ в подвале)

- **Язык интерфейса** — русский / English.
- **Запускать при входе в Windows** — автозапуск (свёрнутым в трей).
- **Сворачивать в трей при закрытии окна** — при включённой опции крестик не
  закрывает программу, а прячет её в область уведомлений (горячие клавиши
  продолжают работать). Полный выход — через меню значка в трее.

## Сборка

```
dotnet build -c Release
```

Требуется .NET 9 SDK (Windows). Готовый exe: `bin\Release\net9.0-windows\Utilities.exe`.

## Технологии

- C# / WPF (.NET 9), `UseWindowsForms` — только ради `NotifyIcon` (трей).
- Настройки: `%LOCALAPPDATA%\SeniorUtilities\settings.json`.
- Без внешних зависимостей. Логика конвертации раскладки — порт из `layoutfix.py`.

## Как добавить новую утилиту

1. Реализуйте логику в папке `Tools/` (по образцу `LayoutFixService.cs`).
2. Добавьте плитку в `MainWindow.xaml` (стиль `TileStyle`) и обработчик в
   `MainWindow.xaml.cs`.
3. Добавьте строки локализации в `Resources/Strings.ru.xaml` и `Strings.en.xaml`.

## Иконка

`app.ico` — белая шестерёнка на зелёном градиенте (стиль Senior Hub),
сгенерирована программно (7 размеров от 16 до 256 px).
